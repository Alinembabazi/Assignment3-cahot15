@ Greenwood Campus Website

A comprehensive campus website demonstrating modern web development techniques and responsive design patterns.

@@ Project Overview

This project showcases a fully functional campus website with multiple sections, navigation features, and interactive elements. The website demonstrates various CSS layout techniques, responsive design principles, and user experience best practices.

@@ Layout Behaviors Demonstrated

 @Navigation System

- Fixed Navigation Bar: Remains accessible while scrolling
- Dropdown Menus: Multi-level navigation with hover effects
- Internal Links: Smooth scrolling to page sections
- Responsive Navigation: Adapts to mobile devices

@ Content Sections

1. Introduction Section: Hero area with campus overview and information list
2. Campus Services: Card-based layout with hover effects
3. Campus Gallery: Multi-column image grid with responsive behavior
4. Announcements: Sticky notices and regular announcement items
5. Display Behaviors: Demonstration of CSS visibility and display properties

@@ CSS Techniques Applied

- Grid Layout: For services, gallery, and demo sections
- Flexbox: For navigation and container layouts
- Positioning: Fixed navbar, sticky announcements
- Transitions: Smooth hover effects and animations
- Responsive Design: Mobile-first approach with media queries
- Semantic HTML5: Proper structure with header, nav, section, footer

@@ Project Files

- index.html: Main HTML structure with semantic elements
- style.css: Complete CSS styling with responsive design
- README.md:Project documentation
